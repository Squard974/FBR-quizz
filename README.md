# Mobile App Project

## Description
Application mobile utilisant Django pour le back-end et React Native pour le front-end.

## Structure du Projet
- `backend/` : Serveur Django et API RESTful.
- `frontend/` : Application mobile React Native.

## Installation
1. Clonez le dépôt.
2. Configurez le backend et le frontend (voir les instructions dans leurs dossiers respectifs).
